## Alerts

**`POST /v1/alerts/rules`**  
Create an alert rule

**`GET /v1/alerts/rules`**  
List alert rules

**`PUT /v1/alerts/rules/{rule_id}`**  
Update a rule (enable/disable, thresholds)

**`DELETE /v1/alerts/rules/{rule_id}`**  
Delete a rule

**`GET /v1/alerts/fired`**  
List fired alert deliveries with trigger details

Rule types: `denial_spike`, `new_source_access`, `isolation_violation`, `high_deny_rate`.

```bash title="Create a denial spike alert"
curl -X POST http://localhost:8000/v1/alerts/rules \
  -H "X-API-Key: apl_yourkey" \
  -H "Content-Type: application/json" \
  -d '{
    "name":             "denial spike",
    "rule_type":        "denial_spike",
    "threshold":        10,
    "window_minutes":   15,
    "cooldown_minutes": 60,
    "webhook_url":      "https://hooks.slack.com/services/..."
  }'
```
